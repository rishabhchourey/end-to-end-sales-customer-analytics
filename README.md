#End-to-End Sales & Customer Analytics

## üìå Project Overview
This project is an end-to-end Sales and Customer Analytics solution designed to analyze business performance using structured data. The goal of this project is to demonstrate the complete data analytics lifecycle, starting from raw data preparation to generating meaningful business insights through analysis and dashboards.

The project uses Excel, SQL, Python, and Power BI to simulate a real-world data analyst workflow. Multiple datasets related to customers, orders, products, payments, and feedback were analyzed to understand sales trends, customer behavior, and payment performance.


## üß© Data Description
The dataset is organized into the following tables:

- **Customers**: Customer details such as name, age, city, and join date  
- **Orders**: Order-level data including product, quantity, price, and order date  
- **Products**: Product categories such as Food, Fitness, Electronics, and Services  
- **Payments**: Payment mode and payment status for each order  
- **Feedback**: Customer ratings and reviews for orders
